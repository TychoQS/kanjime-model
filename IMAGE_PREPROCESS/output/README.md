# Resultados del Preprocesamiento (Output)

Directorio destinado a almacenar los resultados visuales generados por los scripts de preprocesamiento.

## Descripción General

Los resultados se organizan automáticamente en subdirectorios nombrados según el script generador. Cada archivo es una imagen compuesta que muestra la evolución de la imagen original a través de las distintas etapas del pipeline.

## Tabla de Contenidos

| Directorio | Tipo | Descripción |
| :--- | :--- | :--- |
| `custom_binarization/` | Carpeta | Comparativas generadas por `custom_binarization.py`. |
| `README.md` | Doc | Esta documentación. |

## Uso

Este directorio se utiliza para la verificación visual de los algoritmos. Para generar nuevos resultados, ejecute cualquiera de los scripts de la carpeta superior apuntando a una imagen de `samples/`.
