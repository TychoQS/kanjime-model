# Resultados del Preprocesamiento (Output)

Directorio de salida donde se almacenan las imágenes generadas por el pipeline.

## Descripción General

Este directorio contiene las imágenes comparativas generadas por `preprocess.py`. Cada imagen muestra las distintas etapas del preprocesamiento aplicadas a una imagen de entrada.
