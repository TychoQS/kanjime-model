# Inference Test Images

Directory containing handwritten Kanji character images used to test the trained model in real-world scenarios.

## Overview

This directory stores test images external to the training dataset. They are used to validate the model's generalization capability with real handwritten input not seen during training.

## Requirements for New Images

- Images must contain a single centered character.
- Preferably light background with dark strokes (or vice versa).
- Minimum recommended resolution: 128x128 pixels.
- Format: JPEG or PNG.
