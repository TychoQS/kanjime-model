# Fonts

## Overview

This directory contains the typefaces used across the project for visualization.

## Table of Contents

| Name | Type | Description |
| :--- | :--- | :--- |
| `Noto_Sans_JP/` | Directory | Google's Noto Sans JP font for Japanese character rendering. |
| `extract_fonts.sh` | File | Bash script to clone Google Fonts repository and extract Japanese fonts using `fonts_extracter.py`. |
| `extracted_fonts/` | Directory | Destination directory for fonts extracted from the Google Fonts repository. |
| `utils/` | Directory | Contains utility scripts like `fonts_extracter.py` for downloading and filtering Japanese fonts. |
