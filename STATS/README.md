# Estadísticas y Registros (STATS)

## Descripción General

Este directorio centraliza los registros históricos y los resultados cuantitativos obtenidos durante el desarrollo y evaluación del sistema. Alberga los logs de los entrenamientos y los resultados de las experimentaciones.

## Tabla de Contenidos

| Archivo | Tipo | Descripción |
| :--- | :--- | :--- |
| `training_log.csv` | CSV | Registro histórico de experimentos y modelos de entrenamiento (Versionado). |
| `text-detectors-benchmark.csv` | CSV | Resultados de ejecución y medición de rendimiento temporal de modelos detectores de texto. |
| `README.md` | Doc | Esta documentación. |
