# Imágenes de Prueba

Directorio que contiene imágenes de caracteres Kanji manuscritos utilizados para probar el modelo entrenado en escenarios reales.

## Descripción General

Este directorio almacena imágenes de prueba externas al dataset de entrenamiento. Se utilizan para validar la capacidad de generalización del modelo con escritura manuscrita real, no vista durante el entrenamiento.

## Requisitos para Nuevas Imágenes

- Las imágenes deben contener un único carácter centrado.
- Fondo preferiblemente claro con trazo oscuro (o viceversa).
- Resolución mínima recomendada: 128x128 píxeles.
- Formato: JPEG o PNG.
