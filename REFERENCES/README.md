# Referencias y Bibliografía

## Descripción General
Este directorio contiene documentos, artículos, enlaces y cualquier material de referencia utilizado como base teórica o técnica para el desarrollo de este Trabajo de Fin de Grado (TFG). Centraliza la bibliografía para facilitar la consulta y justificación de las decisiones tomadas en el proyecto.

## Tabla de Contenidos
| Archivo | Tipo | Descripción |
| :--- | :--- | :--- |
| `README.md` | Documentación | Guía del directorio de referencias. |

## Uso
Para añadir una nueva referencia a este repositorio, siga estas pautas:
1. Coloque el archivo (si es un PDF o documento) en este directorio o una subcarpeta lógica.
2. Realice un commit utilizando el tag `[REF]` siguiendo el formato establecido en `.antigravity/rules.md`.
3. El commit debe incluir el nombre del autor, el fichero (o URL) y la referencia bibliográfica completa.

## Dependencias
No existen dependencias de software para este directorio, ya que su contenido es puramente informativo y documental.
